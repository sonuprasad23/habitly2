# Habitly Assets

This directory contains asset files for the Habitly app.

## Directory Structure

```
assets/
├── sounds/           # Audio files for feedback
│   ├── habit_complete.mp3
│   ├── habit_uncheck.mp3
│   ├── streak_milestone.mp3
│   ├── timer_complete.mp3
│   ├── timer_tick.mp3
│   ├── goal_complete.mp3
│   └── tap.mp3
├── animations/       # Lottie animation files
│   ├── confetti.json
│   ├── streak_fire.json
│   └── check_mark.json
├── icons/           # Custom icon files
└── fonts/           # Font files
    ├── Outfit-Regular.ttf
    ├── Outfit-Medium.ttf
    ├── Outfit-SemiBold.ttf
    ├── Outfit-Bold.ttf
    ├── Inter-Regular.ttf
    ├── Inter-Medium.ttf
    ├── Inter-SemiBold.ttf
    └── Inter-Bold.ttf
```

## Sound Files

You need to add the following sound files (MP3 format, keep under 100KB each):

1. **habit_complete.mp3** - Short, satisfying "ding" sound (0.3s)
2. **habit_uncheck.mp3** - Soft "whoosh" sound (0.2s)
3. **streak_milestone.mp3** - Celebration jingle (1.5s)
4. **timer_complete.mp3** - Gentle bell sound (2s)
5. **timer_tick.mp3** - Optional soft tick (0.1s)
6. **goal_complete.mp3** - Triumphant fanfare (2s)
7. **tap.mp3** - Light tap feedback (0.1s)

## Font Files

Download fonts from Google Fonts:
- [Outfit](https://fonts.google.com/specimen/Outfit)
- [Inter](https://fonts.google.com/specimen/Inter)

## Animations

Download or create Lottie animations from:
- [LottieFiles](https://lottiefiles.com/)
